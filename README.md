# Logic-Forge-PSET1
Solutions for Logic Forge PSET 1. 
<br>
Includes brute-force and optimized implementations

## Overview
This PSET consists of 6 programming challenges.<br>
Each challenge has a dedicated file for its solution.<br>
The challenges focus on logic building, algorithm design, and problem-solving skills.<br>

## Challenges

1. Team Contribution Multiplier<br>
   Logic to calculate weighted contribution scores for team members.<br>

2. Password Recovery Window<br>
   Determine valid time and conditions to recover a password within constraints.<br>

3. Balanced Performance Score<br>
   Analyze performance metrics and compute balanced scoring values.<br>

4. The Deep Storage Inventory Search<br>
   Logic to efficiently query large inventory data.<br>

5. Fix the Broken Expression<br>
   Parse and correct broken arithmetic or logical expressions.<br>

6. Tower of Hanoi Algorithm<br>
   Classic recursive solution to the Tower of Hanoi problem.<br>

## Commit Strategy
Initial brute-force approaches are committed first.<br>
Optimized solutions are committed later.<br>
Commit history demonstrates problem-solving evolution.<br>
